## 기능
- 로컬 비디오 플레이
  - 내 디바이스에 저장된 비디오를 PiP로 시청할 수 있습니다.
- 네트워크 비디오 플레이
  - 유튜브, 인스타그램 등 웹 사이트에 게시된 비디오를 PiP로 시청할 수 있습니다.
